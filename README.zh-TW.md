## ✨ 功能特點

- 🧮 **多種專業量表** — 收錄多種國際通用的心理健康量表，科學評估心理健康狀態
- 🎨 **紫色漸變設計** — 柔和的紫羅蘭漸變配色，舒緩的視覺體驗
- 🌙 **深色/淺色主題** — 一鍵切換，深夜使用也不傷眼
- 📊 **詳細評分報告** — 給出量化評分及對應的心理健康等級說明
- 📱 **響應式佈局** — 完美適配手機、平板、桌面端
- 🌐 **多語言支援** — 簡體中文、繁體中文、英語、日語

## 🚀 快速開始

```bash
git clone https://github.com/huqingyuan-Python/mindtest.git
cd mindtest
open index.html
```

## 📖 使用說明

1. 開啟頁面，選擇深色或淺色主題
2. 選擇想要進行的心理測評量表
3. 根據自身情況如實填寫每道題目
4. 提交後獲取量化的心理健康評分
5. 查看詳細的結果解讀和建議

> ⚠️ 本工具僅供自測參考，不能替代專業心理健康診斷。如有需要，請尋求專業心理諮詢師的幫助。
