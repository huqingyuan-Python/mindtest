## ✨ Features

- 🧮 **Multiple Professional Scales** — International validated psychological assessment scales for scientific mental health evaluation
- 🎨 **Purple Gradient Design** — Soft violet gradient color scheme for a calming visual experience
- 🌙 **Dark/Light Theme** — One-click toggle, comfortable for late-night use
- 📊 **Detailed Scoring Report** — Quantitative scores with corresponding mental health level descriptions
- 📱 **Responsive Layout** — Perfect experience on mobile, tablet, and desktop
- 🌐 **Multi-language Support** — Simplified Chinese, Traditional Chinese, English, Japanese

## 🚀 Quick Start

### Online Demo

Open [index.html](index.html) directly in your browser to start.

### Local Run

```bash
git clone https://github.com/huqingyuan-Python/mindtest.git
cd mindtest
open index.html
```

## 📖 Usage

1. Open the page and choose dark or light theme
2. Select the psychological assessment scale you want to take
3. Answer each question honestly based on your actual situation
4. Submit to receive your quantified mental health score
5. View detailed result interpretation and recommendations

> ⚠️ This tool is for self-assessment reference only and cannot replace professional mental health diagnosis. If needed, please seek help from a qualified mental health professional.
