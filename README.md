> Mental Health Self-Assessment — 一款专业、科学的心理健康自测工具，帮助你了解自己的心理健康状态，提供多种权威心理量表，支持深色/浅色主题切换。

## ✨ 功能特性

- 🧮 **多种专业量表** — 收录多种国际通用的心理健康量表，科学评估心理健康状态
- 🎨 **紫色渐变设计** — 柔和的紫罗兰渐变配色，舒缓的视觉体验
- 🌙 **深色/浅色主题** — 一键切换，深夜使用也不伤眼
- 📊 **详细评分报告** — 给出量化评分及对应的心理健康等级说明
- 📱 **响应式布局** — 完美适配手机、平板、桌面端
- 🌐 **多语言支持** — 简体中文、繁体中文、英语、日语

## 🚀 快速开始

### 在线体验

直接在浏览器中打开 [index.html](index.html) 即可开始测评。

### 本地运行

```bash
git clone https://github.com/huqingyuan-Python/mindtest.git
cd mindtest
open index.html
```

## 📖 使用说明

1. 打开页面，选择**深色**或**浅色**主题
2. 选择想要进行的心理测评量表
3. 根据自身情况如实填写每道题目
4. 提交后获取量化的心理健康评分
5. 查看详细的结果解读和建议

> ⚠️ 本工具仅供自测参考，不能替代专业心理健康诊断。如有需要，请寻求专业心理咨询师的帮助。
