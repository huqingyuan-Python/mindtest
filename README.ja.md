# 🧠 メンタルヘルス自己診断

> 專門的かつ科学的なメンタルヘルス自己診断ツール — 複数の権威ある心理スケールを採用し、あなたの心理健康状態を評価。ダーク/ライトテーマ切り替え対応。

## ✨ 機能特徴

- 🧮 **複数の専門スケール** — 国際的に通用する心理健康評価スケールを収録
- 🎨 **紫グラデーション設計** — 和らげたいろばんだグラデーション配色
- 🌙 **ダーク/ライトテーマ** — ワンク릭で切り替え、深夜使用も目に優しい
- 📊 **詳細なスコアリングレポート** — 数値スコアと対応する心理健康等级の説明を提供
- 📱 **レスポンシブレイアウト** — モバイル、タブレット、デスクトップに完全対応
- 🌐 **多言語サポート** — 简体中文、繁体中文、英語、日本語

## 🚀 クイックスタート

```bash
git clone https://github.com/huqingyuan-Python/mindtest.git
cd mindtest
open index.html
```

## 📖 使い方

1. ページを開き、ダークまたはライトテーマを選択
2. 受検したい心理評価スケールを選択
3. 各問題に实际情况に合わせて正直に回答
4. 送信して定量化された心理健康スコアを取得
5. 詳細な結果解釈と推奨事项を確認

> ⚠️ このツールは自己診断参考用であり、専門家の心理健康診断に代わるものではありません。必要に応じて、必ず資格のある心理健康専門家にご相談ください。
